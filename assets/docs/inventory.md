# Smart Restock: Inventory Page Manual

The **Inventory Page** is your central hub for tracking and managing all purchased items. It provides a visual overview of your shopping habits, item frequencies, and spending.

## 🔍 Overview
The Inventory Page displays a list of all items you have previously purchased and tracked. Each item is presented in a "Premium Card" that highlights its category, purchase frequency, and average price.

---

## ✨ Key Features

### 1. View Items
*   **Item Cards:** Displays the item name, category, and average price.
*   **Recency Tracking:** Shows how long it has been since you last purchased the item (e.g., "Purchased 3 days ago").
*   **Pull-to-Refresh:** Swipe down on the list to refresh your inventory data from the server.

### 2. Add New Items
*   **Manual Add:** Tap the **+ (Plus)** icon in the top right corner to manually enter a new item into your inventory.
*   **Scan Receipt:** Tap the **Receipt** Floating Action Button (bottom right) to navigate to the scanner and automatically add items from a physical receipt.

## ⚙️ Managing Items

### Edit an Item
To update an existing item:
1.  **Tap** on any item card in the list.
2.  The **Edit Item** dialog will appear.
3.  Modify the name, category, price, or frequency.
4.  Tap **Save** to apply changes.

### Delete an Item
If you no longer wish to track an item:
1.  **Tap** the item card.
2.  Select the **Delete** option within the edit dialog.

---

## 🛠 Filtering & Search

### Filtering by Frequency
You can filter your view to see how often items need restocking:
1.  Tap the **Filter (List)** icon in the AppBar.
2.  Select from:
    *   **All Items**: View everything.
    *   **Weekly**: Items bought every week.
    *   **Bi-Weekly**: Items bought every two weeks.
    *   **Monthly**: Monthly essentials.
    *   **Seasonal**: Items bought once a season.

### Searching (Coming Soon)
*   Tap the **Search (Magnifying Glass)** icon to quickly find specific items by name.
